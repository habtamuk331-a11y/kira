## 🆘 If Something Doesn't Work

### Check These First
1. **Both servers running?** - Check terminal windows
2. **Can access frontend?** - Try http://localhost:5173
3. **Can reach backend?** - Try http://localhost:5000/api/health
4. **Errors in console?** - Check F12 > Console
5. **Database exists?** - Check `server/prisma/dev.db`

### Quick Fixes
```bash
# Restart servers
# (Ctrl+C in each terminal, then npm run dev again)

# Reset database
cd server
rm prisma/dev.db
npm run seed

# Clear browser cache
# Ctrl+Shift+Delete in Chrome
```

See `README.md` troubleshooting section or specific guide files.
